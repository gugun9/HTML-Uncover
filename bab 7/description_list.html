Istilah-Istilah Programming:

Syntax
tata bahasanya sebuah bahasa pemrograman.
Recursion
metode penting yang digunakan di banyak aplikasi. Recursion dapat diartikan sebagai momen di mana ia dapat memerintahkan dirinya sendiri untuk mengulang sebuah proses sampai akhirnya proses tersebut dihentikan oleh kondisi semula.
Data Structures
struktur yang dirancang untuk mengatur, menyimpan dan mengakses data di dalam program kamu.
Algorithm
algoritma adalah langkah-langkah yang digunakan untuk menyelesaikan masalah.
